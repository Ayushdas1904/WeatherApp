# Climax - A Weather Forecasting Website

## Overview

This is a weather forecasting website built with Vite and React. The application allows users to search for weather conditions by city and view real-time weather data. The site features dynamic weather videos that change based on the search results.

## Features

- **Weather Search:** Allows users to input a city and retrieve current weather data.
- **Responsive Design:** Ensures that the website is usable on both desktop and mobile devices.

## Technologies Used

- **React JS:** JavaScript library for building user interfaces.
- **Vite:** Build tool that provides a fast development environment.
- **Weather API:** Fetches real-time weather data.
